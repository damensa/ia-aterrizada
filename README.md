# IA Aterrizada

Landing page premium per a serveis d'Intel·ligència Artificial.

## Característiques

- ✅ Disseny minimalista i modern
- ✅ Selector d'idiomes (Català / Castellano)
- ✅ Totalment responsive
- ✅ Optimitzat per rendiment

## Tecnologies

- HTML5
- CSS3 amb variables CSS
- JavaScript vanilla

## Instal·lació

Simplement obre `index.html` al teu navegador.

## Idioma

La pàgina suporta dos idiomes:
- Català
- Castellano

El selector d'idiomes es troba a la part superior dreta i guarda la preferència de l'usuari.
